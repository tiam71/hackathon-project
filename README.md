# 吾幫齡-黑客松競賽

## 定位
「吾幫齡」是一個面向專業照顧人員、非專業照顧人員、長者及其家屬的綜合性服務平台。  
透過 **職缺發布、履歷管理、醫療設備商城及交通資訊整合** 等功能，提供一站式照護解決方案。

## 目標
- 因應全人醫護人力短缺與長照需求增加，建立一個鄰里互助為導向的平台。
- 協助長照需求者快速媒合合適的照顧人員。
- 結合定位系統，進行地理位置與工作內容的即時媒合。
- 整合認證篩選機制，提供長者及其家屬可信的照顧人員與醫療設備管道。
